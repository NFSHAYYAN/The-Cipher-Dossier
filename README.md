# 🛡️ The Cipher Dossier 
A comprehensive C++ suite for encryption, decryption, and secure key exchange.

## 🚀 Overview
This project demonstrates the implementation of both classical and modern cryptographic algorithms. It is designed with a focus on mathematical accuracy and computational efficiency.

## 🛠️ Features
- **Classical Ciphers:** Caesar & Playfair (Rule-based substitution).
- **Asymmetric Encryption:** RSA implementation optimized with **Chinese Remainder Theorem (CRT)** for faster decryption.
- **Key Exchange:** Diffie-Hellman protocol for secure shared secret generation.
- **UI/UX:** Interactive CLI with color-coded feedback and ASCII art.

## 🧪 Mathematical Implementations
- Primality testing for secure key generation.
- Modular exponentiation (PowMod).
- Extended Euclidean Algorithm for modular inverses.

## 📂 File Structure
- `main.cpp`: Core application logic and menu system.
- `CIPHER.h`: Header file containing function prototypes and color definitions.
